#How to use the ZDS skill?
The skill encodes everything Claude needs to build on-brand Zero Design System artifacts without requiring the Figma MCP. It auto-triggers on phrases like "Zero design system", "Zero Foundation", "gb-teal", or any request to build a Zero-branded UI.

##For Claude Desktop app
<p>1. Click on **Customize** in the sidebar
<br>2. Click on **Skills** > **+** > **Upload a skill** > Upload the `zero-design-system.skill` file
<br> 3. Mention 'zero design system' in your prompt
</p>

##For Claude CLI (Terminal)
<p>1. Set up these folders: `~/.claude/skills`
<br>2. Paste the `zds` folder into the skills folder
<br>3. Start your prompt with /zds or mention 'zero design system' in your prompt
