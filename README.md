# How to use the ZDS skill?
The skill encodes everything Claude needs to build on-brand Zero Design System artifacts without requiring the Figma MCP. It auto-triggers on phrases like "Zero design system", "Zero Foundation", "gb-teal", or any request to build a Zero-branded UI.

## Downloading the files
1. Click on the green **Code** button on the top right > **Download ZIP**

## Setting up for Claude Desktop app
1. Click on **Customize** in the sidebar
2. Click on **Skills** > **+** > **Upload a skill** > Upload the `zero-design-system.skill` file
3. Mention "zero design system" in your prompt

## Setting up for Claude CLI (Terminal)
1. Set up the skills folder in your preferred folder: `~/.claude/skills`
    * e.g. `Documents/.claude/skills`
2. Paste the `zds` folder into the `skills` folder
3. Start your prompt with `/zds` or mention "zero design system" in your prompt
